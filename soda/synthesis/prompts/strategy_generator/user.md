## Strategy Requirements

### 1. Strategy Overview
For each strategic approach:
- **Strategy Name**: Clear, descriptive title
- **Strategic Pattern**: DIFFERENTIATED/DISRUPTIVE/DOMINANT/SUSTAINING
- **Target Segments**: Which segments this strategy serves
- **Core Rationale**: ODI evidence supporting this approach

### 2. Implementation Plan
For each strategy:
- **Priority Actions**: Top 3-5 initiatives based on opportunity scores
- **Resource Allocation**: How to distribute budget across opportunities
- **Timeline**: 6-month phases with key milestones
- **Success Metrics**: How to measure progress

### 3. Opportunity Addressing
For each strategy:
- **Underserved Focus**: How strategy addresses highest opportunity scores
- **Overserved Management**: How to handle over-investments
- **Table Stakes Protection**: Maintaining critical service levels
- **Cross-Segment Considerations**: Managing segment interactions

### 4. Risk Mitigation
For each strategy:
- **Implementation Risks**: Key challenges and mitigation approaches
- **Constraint Management**: How to work within business limitations
- **Contingency Planning**: Alternative approaches if primary plan fails

## Format Requirements
- Start each strategy with clear executive summary
- Use specific opportunity scores as evidence
- Reference business constraints explicitly
- Provide concrete, measurable actions
- Include decision criteria for choosing between strategies

Focus on actionable recommendations that maximize opportunity value within business constraints.
